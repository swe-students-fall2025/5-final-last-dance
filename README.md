### Requirements:
- Python 3.12+
- MongoDB 4.0+
- pipenv

### Installation

##### Set up environment
edit .env.example .env

Then install dependencies and run
`pipenv install`
`pipenv shell`
`python app.py` or `flask run`


Visit: http://127.0.0.1:5000
